# Current State

This folder records implementation progress for the generated template.

Stable docs under `docs/architecture`, `docs/platform`, `docs/modules`, and
`docs/testing` define durable intent and rules. Current-state docs may describe
what the template currently ships, which smoke surfaces exist, and which pieces
are intentionally absent at this point.

## Index

- [server.md](server.md) records the shipped backend slice.
- [web.md](web.md) records the shipped frontend slice.
- [platform.md](platform.md) records the shipped local platform, auth,
  observability, and testing surfaces.
- [identity.md](identity.md) records the shipped Identity module slice.
- [../modules/products.md](../modules/products.md) records the durable intent
  for the shipped Products module slice.
